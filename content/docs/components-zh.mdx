## LinkButton

链接按钮组件，用于链接 URL 并显示文本。

```mdx
<LinkButton url="https://github.com/showlotus/fumadocs-template" text="Github" />
```

<LinkButton url="https://github.com/showlotus/fumadocs-template" text="Github" />

## ThemeImage

图片组件，用于在亮色和暗色模式下显示不同的图片。

```mdx
<ThemeImage light="./imgs/sun.svg" dark="./imgs/moon.svg" className="w-10" />
```

点击 <ToggleTheme /> 切换主题，图标会随之变化。

<ThemeImage light="./imgs/sun.svg" dark="./imgs/moon.svg" className="w-10" />

## CodeBlock

代码块组件，用于显示代码块。

{/* prettier-ignore */}
```mdx
<CodeBlock lang="tsx" code={`'use client'

import Demo from './components/demo.tsx'

export default function Foo() {
    return <div>This is a Foo with <Demo /></div>
}`} />
```

<CodeBlock lang="tsx" code={`'use client'

export default function Foo() {
    return <div>This is a Foo with <Demo /></div>
}`} />

## ViewCode

既能渲染指定组件也能查看其源代码的组件。

```mdx
<ViewCode src="./components/demo.tsx" />
```

<ViewCode src="./components/demo.tsx" />

或者使用 `view-code` 关键字来注入到代码块中：

<CodeBlock lang="tsx" code={`\`\`\`mdx view-code
'use client'

import Demo from './components/demo.tsx'

export default function Foo() {
    return (
      <div className="border-4 border-solid rounded p-2">This is a Foo 
        <div className='border-4 border-solid rounded p-2'>
          <Demo />
        </div>
      </div>
    )
}
\`\`\``} />

```mdx view-code
'use client'

import Demo from './components/demo.tsx'

export default function Foo() {
  return (
    <div className="border-4 border-solid rounded p-2">This is a Foo 
      <div className='border-4 border-solid rounded p-2'>
        <Demo />
      </div>
    </div>
  )

}
```
